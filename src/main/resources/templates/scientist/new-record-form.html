➕ Neuen Datensatz vorschlagen

Schlagen Sie einen neuen CO2-Datensatz vor. Ihre Eingabe wird vom Administrator geprüft.

ℹ️ Hinweis
Ihr Vorschlag wird zunächst gespeichert und muss vom Administrator genehmigt werden, bevor er in die Datenbank übernommen wird. Bitte geben Sie eine aussagekräftige Begründung für Ihren Vorschlag an.
Wählen Sie das Land für den neuen Datensatz
Geben Sie das Jahr ein (z.B. 2023)
Geben Sie den CO2-Wert in Tonnen ein (z.B. 1234.56)
VERIFIED = Verifizierte Daten | ESTIMATED = Geschätzte Daten | PENDING = Ausstehend
Eine aussagekräftige Begründung erhöht die Chance auf Genehmigung
